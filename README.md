# TBB - The Best Bet
The Best Bet is an odds comparator (chance of an event occurring) for the main sports betting websites operating in Brazil, aiming to reduce losses and maximize bettors' gains.

#  Features
1. Odds finder 
2. Odds comparator
3. "Sure profit"
